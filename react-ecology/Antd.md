> `antd` 是基于 Ant Design 设计体系的 React UI 组件库，主要用于研发企业级中后台产品。
>
> [Antd](<https://ant.design/docs/react/introduce-cn>)



# 安装

## 使用 npm 或者 yarn 安装

~~~shell
# npm 安装
$npm install antd --save


# yarn 安装
$yarn add antd
~~~

## 引入

~~~javascript
// 需要全局引入 css 样式
import 'antd/dist/antd.css'

// 组件则可以按需引入
~~~
